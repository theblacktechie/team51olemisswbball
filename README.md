# What They Gave · Team 51 · Ole Miss WBB 2025-26

A season retrospective and love letter to the Ole Miss Women's Basketball Class of 2026.

**Live at:** `https://theblacktechie.github.io/team51olemisswbball/`

---

## Deploy to GitHub Pages

1. Create a new repository on GitHub (e.g. `olemisswbb-team51`)
2. Upload `index.html` to the root of the repository
3. Go to **Settings > Pages**
4. Under **Source**, select `main` branch and `/ (root)`
5. Click **Save** — your site will be live within a minute

---

## What's Inside

Five tabbed acts, fully self-contained in a single HTML file. No dependencies, no build step, no external assets beyond the Inter font (loaded from Google Fonts).

| Act | Title | Content |
|-----|-------|---------|
| I | Give | Season word, stats strip, revolution grid, Coach Yo quotes |
| II | The Arc | 12-card season timeline including Winter Storm Fern and Birmingham games |
| III | Seven Gifts | All 7 seniors in jersey order (#3 through #32), Cotie featured |
| IV | 0.8 Seconds | Scoreboard, quarter breakdown, four moment blocks, key stats, postgame quotes |
| V | Hotty Toddy | Full love letter, officiating context, closing statement |
